# PROJECT SETUP

## Objective -
Set up a basic Express.js application from scratch and understand the essential project structure

---

## Topics Covered -

- Initializing a Node.js project
- Understanding `package.json` as well as `package-lock.json` and diff between them
- Installing Express
- Creating an Express server
- Defining routes
- Running the application
- Introduction to `.env`
- Understanding `.gitignore`
- Observing Deployment through Digital Ocean

---

## Commands Used -

### Initialize project

```bash
npm init
```

Creates a `package.json` file.

### Install Express & dotnenv

```bash
npm install express
npm i dotenv
```

### Run application

```bash
npm run start
```

---

## Folder Structure

```
project/
│
├── package.json
├── package-lock.json
├── .env
├── .gitignore
└── index.js
```
